# Skor Toplama Oyunu (C#)

Bu proje, C# ile yazılmış basit bir konsol tabanlı oyun. Oyuncu, düşen nesneleri ('*' ve 'O') yakalayarak skor toplar.

## Özellikler
- Konsol tabanlı oyun
- Rastgele düşen nesneler
- Skor sistemi
- Log dosyası (log.txt)

## Kontroller
- Sol ok: Sola hareket
- Sağ ok: Sağa hareket
- ESC: Oyunu bitir

## Çalıştırma
1. .NET SDK yüklü olduğundan emin olun.
2. Projeyi klonlayın veya indirin.
3. Terminalde proje klasörüne gidin.
4. `dotnet run` komutunu çalıştırın.

## Log
Oyun sırasında tüm hareketler ve olaylar `log.txt` dosyasına kaydedilir.
